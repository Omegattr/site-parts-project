HTML

ハイパーテキスト・マークアップ・ランゲージ（HTML）は、ウェブブラウザで表示される文書の標準マークアップ言語です。カスケーディング・スタイル・シートなどの技術や、JavaScriptなどのスクリプト言語によってサポートされます。

CSS

カスケーディング・スタイル・シート（CSS）は、HTMLなどのマークアップ言語で記述された文書の表示方法を記述するために使用されるスタイルシート言語です。CSSは、HTMLやJavaScriptと並んで、ワールド・ワイド・ウェブの基盤となる技術です。

JAVASCRIPT

JavaScript（略称JS）は、HTMLやCSSと並んで、ワールドワイドウェブの中核技術の一つであるプログラミング言語です。ウェブサイトの97%以上が、ウェブページの動作にクライアント側でJavaScriptを使用しており、多くの場合サードパーティ製のライブラリが組み込まれています。

REACT

Reactは、UIコンポーネントに基づいてユーザーインターフェースを構築するための、無料のオープンソースのフロントエンドJavaScriptライブラリです。Metaと、個人の開発者や企業で構成されるコミュニティによってメンテナンスされています。

NODE

Node.jsは、V8エンジン上で動作し、Webブラウザ外でJavaScriptコードを実行する、オープンソースでクロスプラットフォームなバックエンドJavaScriptランタイム環境です。
